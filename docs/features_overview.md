# 功能概览 (Feature Overview)

本文档用于记录 **Wanderlust's Way To Survive** 模组当前已实现的功能及其简要说明。

## 1. 旅者附魔 (Traveler Enchantment)
- **描述**: 提供移动速度加成的附魔系统。
- **机制**:
  - 速度加成取决于经验等级和附魔等级。
  - 支持配置特定方块上的速度上限 (例如土径、灵魂土、浮冰)。
  - 基础速度和倍率可通过配置文件调整。

## 2. 苦力怕机制 (Creeper Scaling)
- **描述**: 基于苦力怕生命值的动态爆炸威力。
- **机制**:
  - **满血状态**: 爆炸威力减弱 (默认 0.5 倍)。
  - **残血状态**: 爆炸威力大幅增强 (默认 2.5 倍)。
  - 增加了与苦力怕战斗的风险与策略性。

## 3. 马匹增强 (Horse Mechanics)
- **穿叶而行**: 当马匹踩在非树叶方块上时，骑乘者和马匹头部可以穿过树叶，避免在森林中卡住。
- **深水游泳**: 骑乘马匹进入深水时，马匹会自动上浮，支持在水中骑乘。
- **第一人称透明化**: 骑乘时低头看，马匹模型会自动变透明，减少视野遮挡。
- **号角召唤**:
  - 使用山羊角可以绑定坐骑。
  - 再次使用山羊角可以将绑定的坐骑召唤到身边。
  - 召唤伴随粒子特效。

## 4. 船只旗帜 (Boat Banner)
- **描述**: 允许在船上插旗帜。
- **机制**:
  - **装饰**: 旗帜会渲染在船尾。
  - **加速**: 插旗的船只可获得速度加成 (默认 1.2 倍)。
  - 支持详细的渲染位置、旋转和缩放配置。

## 5. 骑乘末影珍珠 (Mounted Pearl Teleport)
- **描述**: 优化骑乘时的传送体验。
- **机制**:
  - 当骑乘者投掷末影珍珠时，坐骑会跟随玩家一同传送到落点。
  - 解决了原版传送导致人马分离的问题。
